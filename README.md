# Mistral Chat Desktop

App desktop leve que abre o [Mistral Chat](https://chat.mistral.ai) em uma janela limpa, sem abas de navegador.

- ✅ **Login automático** — usa seus cookies do navegador, sem precisar logar de novo
- ✅ **Multiplataforma** — Windows, Linux e macOS
- ✅ **20+ navegadores** — Chrome, Edge, Brave, Firefox, Opera, Vivaldi e mais
- ✅ **Código aberto** — Python puro, ~10 MB
- ✅ **Sem telemetria** — não coleta nenhum dado

## Download

| Plataforma | Download |
|------------|----------|
| **Windows** | [`Mistral-Setup.exe`](https://github.com/Ryanabcraft/mistral-desktop/releases/latest) — Instalador Inno Setup |
| **Linux / macOS** | [`mistralchat_app.py`](mistralchat_app.py) — Rode com `python3 mistralchat_app.py` |

## Como usar

### Windows
1. Baixe o instalador `.exe` da [última release](https://github.com/Ryanabcraft/mistral-desktop/releases/latest)
2. Execute e siga os passos
3. Pronto! O Mistral Chat abre em janela limpa

### Linux / macOS
```bash
wget https://raw.githubusercontent.com/Ryanabcraft/mistral-desktop/master/mistralchat_app.py
python3 mistralchat_app.py
```

Requer Python 3.6+ e um navegador instalado.
